This browser is missing key features!

Typeboard was built from the ground up to experiment with modern features on the web such as CSS variables and flex layouts. Unfortunately this browser does not support some of these features.

The good news is that most browsers do, including Chrome, Safari, and Firefox. Learn more at CanIUse.com.

If you are feeling adventurous and want to continue, be aware that key features will break. Good luck!
